Cómo arrancar la clase 2

Una sola línea para clonar el ejercicio:

git clone https://github.com/dpetrocelli/diplo-unq-blockchain-clase2.git
cd diplo-unq-blockchain-clase2
forge install foundry-rs/forge-std --shallow
forge build
forge test

Tienen que ver 10 passed; 0 failed. Después abrí la guía de la clase 2.
